Flexbox is the new layout of CSS(3). CSS(3) flexbox is used to display elements according to a predetermined design when the page layout is displayed on different screen sizes and different devices. A flexbox consists of a flex container and flex content. To convert a container to flex, set the element's display property as value flexor inline-flex . A flex container can contain one or more flex contents. Flex content resides inline inside the flex container. By default each flex container has one line.

Bootstrap

Bootstrap is the most popular HTML, CSS, and JavaScript framework for developing responsive and mobile fast websites. Bootstrap is very easy to use but only if you know HTML, CSS you can use Bootstrap. So it is important to know HTML before using Bootstrap. Typography, forms, buttons, tables, navigation, modals, image carousels and many more can be used without the hassle of coding through Botstrap. After designing the website with HTML and CSS, it is responsive to make it suitable for all sizes on mobile, tablet, PC. Bootstrap is compatible with all modern browsers such as Chrome, Firefox, Internet Explorer, Safari, and Opera

Tailween

Tailwind CSS is a CSS framework. It can easily create custom user interface. It is a Utility-first CSS Framework which means that custom designs can be created with it through utility classes, without having to follow the rules of normal CSS. Moreover, with Tailwind CSS it is possible to build UI in very short time
